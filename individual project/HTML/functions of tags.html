TAG FUNCTION LIST

  • Tags are arranged in alphabetical order.

  • The list provides some common tags.

Tag

Function
< !--...-- > Annotation tags are used to insert notes into the source code, The annotation will not appear in the browser.
< a > Define hyperlinks to link from one page to another page,the most important attribute of the tag is the href property, which indicates the target of the link.
< body > Defining the main content of a document, the tag contains all the contents of the document (such as text, hyperlinks, images, tables, lists, and so on).
< br > Insert a simple newline character,the tag is empty tag (meaning it doesn't have end tag, so it's wrong: < br >< /br >).
< div > Define partitions or sections in a document (division/section), which can separate documents into separate and different parts.
< h1 > to < h6 > < h1 > - < h6 > tags define headings. < h1 > defines the largest title, < h6 > defines the smallest title.
< head > Defines the head of a document, which is the container for all head elements. The elements in the tag can refer to scripts, indicating where the browser can find the stylesheet, provide meta information, and so on.
< html > Tells the browser that it is an HTML document itself. The < html > and < / html > tags define the beginning and end points of the document, which is the head and subject of the document between them.
< img > Embedding an image into a web page.The tag have two essential attributes: the src attribute and the alt attribute:src set the URL of the display imageand alter alternative text for prescribed images.
< li > Define list items that can be used in ordered lists (< ol >) and unordered lists (< ul >).
< link > Defining the relationship between a document and external resources, the most common use of tags is linking style sheets.
< ol > Define a sequence table.
< p > Defines a paragraph.The tag automatically creates some gaps before and after the browser automatically adds these spaces, and you can also specify the style sheet.
< script > Insert a JavaScript in the HTML page.
< span > Used to combine inline elements in a document.
< strong > Define the text as a more intense and stressed content.
< style > Defining style information for HTML documents. In the tag, you can specify how to render the HTML document in the browser. The type property is required, and the only possible value is "text/css". The tag is in the head section.
< table > Define HTML table. The HTML table consists of table elements and one or more tr, th, or td elements: tr elements define table rows, th elements define headers, and td elements define table units.
< ul > Define unordered list.
