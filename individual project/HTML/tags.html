Tag syntax

  1. Tags are enclosed by English brackets, "<" and ">", such as < html > is a label.

  2. Tags in HTML generally appear in pairs, beginning with tags and ending tags. The end tag is one more than the start tag. Like:
    (1)< p >< / p >
    (2)< div >< / div >
    (3)< span >< / span >

  3. Tags and tags can be nested, but the order must be consistent, such as: nested < p > in < div >, then < / p > must be placed in front of < / div >. As shown in the following picture.For example:
    < div >< p >hello world!!!< / p >< / div >

  4. HTML tags are not case sensitive, < h1 > and < H1 > are the same, but lowercase is recommended, because most programmers take lowercase as the norm.

Basic structure of HTML file

A HTML file has its own fixed structure:
< html >
< head >...< / head >
< body >...< / body >
< / html >
 Analysis:
  1. < html >< / html > is called root tag, and all web tags are in < html >< / html >.

  2. The < head > tag is used to define the head of the document, which is the container for all head elements.

  3. The content between the < body > and < / body > tags is the main content of the web page, such as < h1 >, < p >, < a >, < img > and other web content tags, where the contents of the tag will be displayed in the browser.
