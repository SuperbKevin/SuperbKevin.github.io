CSS (English full name: Cascading Style Sheets) is a computer language used to represent HTML. CSS can not only modify the webpage statically, but also dynamically format all the elements of the page with a variety of scripting languages.

CSS can control the layout of elements in web pages precisely, support almost all font size styles, and have the ability to edit web objects and model styles.

Characteristic

CSS provides a style description for HTML markup language, and defines the display mode of its elements. CSS is a breakthrough in the field of Web design. It can be used to modify a small style and update all the page elements associated with it.

Working principle

CSS can define style structures, such as fonts, colors, locations, and so on, which are used to describe information formatting and reality in Web pages. CSS styles can be stored directly in HTML pages or separate style sheets. A style rule consists of one or more style attributes and their values.
  • When used externally, the style rule is placed in an external stylesheet document with a file extension .Css.

  • The internal style sheet is directly placed in the page, and the external style sheet is kept in an independent document. The web page links the external style sheet through a special label.
